## 三、SDK 限制项及降级方案

### 1. frontTrafficInfo（前方路况查询）

- **现象**：导航中调用返回"未规划路线"或 timeout
- **原因**：SDK 的 frontTrafficInfo 需要特定的导航启动方式，aroundSearch+select 和 naviToPoi 均不触发
- **降级方案**：使用高德 distance API 查剩余距离/时间
- **对应语料**："还有多远"、"还有多久到"、"前面限速多少"
- **SKILL.md 状态**：已更新，标注用高德 API 替代

### 2. selectRoute（路线切换）

- **现象**：返回 SDK error 10050 "抱歉暂时帮不了你了"
- **原因**：aroundSearch 只规划单条路线，没有多路线可切换
- **降级方案**：用高德 direction strategy=10 获取3条路线对比，选择最优路线后重新 naviToPoi
- **对应语料**："换条路"、"走第2条路线"
- **SKILL.md 状态**：已文档化替代方案

### 3. addFavourite（收藏当前位置）

- **现象**：返回"非法参数"
- **原因**：SDK 要求在地图 POI 详情页上下文中调用
- **降级方案**：用 navHistory 的 favorite 功能替代收藏
- **对应语料**："收藏这里"
- **SKILL.md 状态**：已标注限制，提供替代方案
