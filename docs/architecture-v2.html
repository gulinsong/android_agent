关键改进点：
① 省去 QQ 服务器中转（Webhook 延迟 + 网络往返）≈ 2-3s
② 省去 SILK ↔ WAV 双向转码（输入 + 输出）≈ 1-3s
③ 省去 QQ files API 上传等待 ≈ 1-2s
④ STT 流式识别（边说边转）≈ 省去 0.5-1s
⑤ TTS 流式播放（边生成边播）≈ 体感再降 0.5-1s
⑥ LLM 仍是最大瓶颈（~4.5s），未来可用 车机 NPU 小模型 降至 <1s
