# 待解决功能清单

对照完整语料需求，梳理当前 skill 未覆盖的功能，按原因分类，便于后续逐项突破。

**已覆盖**的功能不在此文档中，详见 car-nav/SKILL.md 和 car-poi/SKILL.md 的决策指南。

## 一、需要感知能力（摄像头/雷达/ADAS）

以下需求需要实时视觉感知或车辆传感器数据，**完全超出 skill 和 API 层面的能力**，需要 ADAS 或车载视觉系统支持。

### 1. 车道级感知

| 需求 | 示例语料 |
|------|---------|
| 当前车道信息 | "我现在在哪个车道上？" |
| 车道类型 | "当前车道是直行车道还是转弯车道？"、"这个车道是左转专用道吗？" |
| 车道属性 | "这个车道是公交专用道吗？"、"当前车道是可变车道吗，现在是什么方向？" |
| 车道是上/下坡 | "前面的车道是上坡还是下坡？"、"这条车道的坡度是多少度？" |
| 车道大型/小型 | "当前车道是大型车道还是小型车道？" |
| 上高架专用车道 | "这条车道是上高架的专用车道吗？" |
| 车道变窄 | "前面车道是不是变窄了？" |
| 下穿隧道车道 | "这个车道是下穿隧道的车道吗？" |
| 车道前方去向 | "我这个车道的前方通向哪里？"、"当前车道能直接上高速吗？" |
| 车道即将变化 | "前面500米，我这个车道会变成左转专用道吗？" |
| 潮汐车道 | "这条车道是潮汐车道吗，现在的行驶方向是什么？" |
| 能否掉头 | "我这个车道能掉头吗？" |

**结论**：需要接入车载 ADAS 的车道识别 API，或接入高精度地图车道数据。短期无解。

| 标线颜色/内容 | "这个路口地上的标线是白色的还是黄色的？"、"地面有没有写限速数字？" |
| 导向箭头 | "路面上的导向箭头是指向哪里的？"、"这个路口地面有左转箭头吗？" |

## 三、LLM 可部分覆盖

以下需求高德 API 无直接数据，但 LLM 知识可给出合理回答（需注明参考）。

### 10. LLM 知识直接回答

| 需求类型 | 示例语料 | 策略 |
|---------|---------|------|
| 道路属性 | "深圳沿江高速全长多少公里？"、"深南大道有多宽？" | LLM 知识回答，注明"供参考" |
| 交通规则 | "滨海大道限速多少？"、"外地牌照能走吗？" | LLM 知识 + 注明"以实际标志为准" |
| 限行政策 | "今天限行吗？" | LLM 知识 + 当日日期判断 |
| 场景推荐 | "情人节适合情侣散步的景点"、"团建找农家大院" | LLM 知识提候选 → 高德验证 |
| 地标描述搜索 | "白色圆形巨型建筑" → 深圳湾文化广场 | LLM 识别 → geocode 验证 |
| 商家优惠 | "这家店有没有优惠？" | **已实现**：LLM 知识回答 + /browse 打开美团 |
| 排队/等位 | "海底捞要等位多久？" | **已实现**：基于时间段预测 + /browse 打开美团 |
